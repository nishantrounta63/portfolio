## üì• Installation

### Prerequisites
- Web browser (Chrome, Firefox, Safari, Edge)
- Basic understanding of HTML, CSS, and JavaScript (for customization)

### Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/portfolio.git
   cd portfolio
   ```

2. **Open the project**
   - Open `index.html` in your web browser
   - Or use a local server:
     ```bash
     # Using Python
     python -m http.server 8000
     
     # Using Node.js
     npx http-server
     ```

3. **Access the website**
   - Open your browser and navigate to `http://localhost:8000`

## üé® Customization

### Personal Information

1. **Update HTML (`index.html`)**
   - Replace name, email, phone number
   - Update section content
   - Modify project descriptions
   - Add your own images

2. **Update CSS (`css/style.css`)**
   - Change color scheme in `:root` variables
   - Modify animations and transitions
   - Adjust responsive breakpoints
   - Customize fonts

3. **Update JavaScript (`js/main.js`)**
   - Modify typing animation roles
   - Update form submission handler
   - Customize particle settings
   - Add analytics tracking

### Color Scheme

The color palette can be customized in `css/style.css`:

```css
:root {
    --primary-color: #6366f1;      /* Primary blue-purple */
    --secondary-color: #8b5cf6;    /* Secondary purple */
    --accent-color: #ec4899;       /* Accent pink */
    --dark-bg: #0f172a;            /* Dark background */
    --darker-bg: #020617;          /* Darker background */
}
```

### Fonts

Current fonts:
- **Poppins** - Main font
- **Fira Code** - Code font

To change fonts, update the Google Fonts link in `index.html` and CSS variables.
